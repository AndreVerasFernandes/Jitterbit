# Jitterbit API Skeleton

Esqueleto de uma API em Node.js usando Express (JavaScript).

Instalação

```bash
npm install
```

Rodando

```bash
npm run dev   # usa nodemon
npm start     # inicia em modo production
```

Estrutura

- `package.json` - dependências e scripts
- `src/index.js` - entrada do servidor
- `src/routes/` - rotas
- `src/controllers/` - controladores
- `src/middlewares/` - middlewares (ex: tratamento de erro)
